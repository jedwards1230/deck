# deck

A terminal slide presenter

---

## Features

<!-- pause -->

- Markdown rendering via Glamour

<!-- pause -->

- Progressive reveal with pauses

<!-- pause -->

- Vim-style navigation

---

## Code Execution

Press `ctrl+e` to run this:

```bash
echo "Hello from deck!"
date
```

## Navigation

| Key | Action |
|-----|--------|
| `h/l` | Prev / Next |
| `j/k` | Prev / Next |
| `gg` | First slide |
| `G` | Last slide |
| `3G` | Slide 3 |
| `/` | Search |
| `ctrl+e` | Run code |
| `y` | Yank code |
| `q` | Quit |
